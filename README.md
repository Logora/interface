# Logora UI

---
Design system for [Logora](https://logora.fr) made with React and using [Bit](https://bit.dev)

View all components [here](https://bit.cloud/logora/debate/)


## Install

```bash
npm i -g @teambit/bvm
bvm install
```

## Run

```bash
bit start
```

## Test components

```bash
yarn test
```
