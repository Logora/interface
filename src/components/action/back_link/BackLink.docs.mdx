import { BackLink } from './BackLink';

A button that opens a link with back arrow and text.
Props can be passed to the underlying [Link](https://bit.cloud/logora/debate/action/link) component to have an external link or a link from react-router.

### Component usage

```js
import BackLink from '@logora/debate.action.back_link';
```
```js
<BackLink
    to="https://example.com/source-link"
    text="Back to article"
/>
```
