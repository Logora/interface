import { Button } from './Button';

A component that renders a clickable button. The button comes with different styles and the possibility to add icons.

### Component usage

To import the Button component:

```js
import { Button } from '@logora/debate.action.button';
```

To use the Button component:

```js
<Button 
    active={true} 
    leftIcon={<i>Left Icon</i>}
    rightIcon={<i>Right Icon</i>}  
    accent="success"
    handleClick={clickCallbackFunction} 
    className={classNameToPass}
>
    Click !
</Button>
```

### Properties

| Prop name        | Type     | Required | Default  | Description                                                |
| ---------------- | -------- | -------- | -------- |----------------------------------------------------------- |
| `type`           | string   | No       | 'button' | The type of the button (e.g. 'button', 'submit').          |
| `active`         | boolean  | No       | `false`  | The active state of the button.                            |
| `leftIcon`       | node     | No       | -        | The left icon of the button.                               |
| `rightIcon`      | node     | No       | -        | The right icon of the button.                               |
| `accent`         | string   | No       | -        | The color accent of the button, can be `success` or `danger`.    |
| `border`         | boolean  | No       | true     | Will not show the border if false.                         |
| `handleClick`    | function | No       | -        | A callback function for the onClick event.        |
| `className`      | string   | No       | -        | A custom classname.                         |
| `children`       | node     | No       | -        | The content of the button.                              |
