import { Button } from './Button';

A component that renders a clickable button. The button comes with different styles and the possibility to add icons.

### Component usage

To import the Button component:

```js
import { Button } from '@logora/debate.action.button';
```

To use the Button component:

```js
<Button 
    active={true} 
    leftIcon={<i>Left Icon</i>}
    rightIcon={<i>Right Icon</i>}  
    accent="success"
    handleClick={clickCallbackFunction} 
    className={classNameToPass}
>
    Click !
</Button>
```
