import { GoogleLoginButton } from './GoogleLoginButton';
import { MemoryRouter } from "react-router";

A button component to login to Google with OAuth2.

When clicking on the button, a popup will appear that will ask the user to login to Google. User will be
redirected to _redirectUri_ afterwards if login is successful, with authorization code.

Two scopes are requested :
- https://www.googleapis.com/auth/userinfo.email 
- https://www.googleapis.com/auth/userinfo.profile

### Component usage

```js
import { GoogleLoginButton } from '@logora/debate.auth.google_login_button';
```

```js
    <MemoryRouter>
        <GoogleLoginButton 
            text={"Sign in with Google"}
            googleClientId={"google-client-id"}
            redirectUri={"https://auth.redirect/}
            onCode={(code) => console.log(code)}
            onClose={closeCallbackFunction}
        />
    </MemoryRouter>
```
