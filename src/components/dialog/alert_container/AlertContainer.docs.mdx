import { AlertContainer } from './AlertContainer';

This component provides a toaster container which enables to display alerts at the bottom center of the screen.
It is using [react-hot-toast](https://react-hot-toast.com/).

Use the `toast` function from react-hot-toast to display a toast.

### Component usage
```js
import AlertContainer from '@logora/debate.dialog.alert_container';
```
```js
<AlertContainer />
```

```js
import { toast } from 'react-hot-toast';

toast("My message !");
```
