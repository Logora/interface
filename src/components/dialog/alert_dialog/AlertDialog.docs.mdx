import { AlertDialog } from './AlertDialog';

This component should be used with react-hot-toast and AlertProvider.

It displays a message, and optionally points earned by the user.

### Component usage
```js
import { AlertDialog } from '@logora/debate.dialog.alert_dialog';
```
```js
<AlertDialog 
    text={"Your text"}
    points={34}
    variant={"success"}
/>
```
