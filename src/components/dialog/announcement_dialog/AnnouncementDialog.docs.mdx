A component that shows a message with icon. Icon and content can be customized.

It should be inside [IconProvider](https://bit.cloud/logora/debate/icons/icon_provider).

### Component usage

```js
import { AnnouncementDialog } from '@logora/debate.dialog.announcement_dialog';
```

```js
<AnnouncementDialog 
    icon={Icon}
    message={"Your text"}
    fullWidth={true}
    className={classNameToPass}
    iconClassName={iconClassNameToPass}
/>
```
