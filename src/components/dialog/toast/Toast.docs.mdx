---
description: A component to display a toast dialog.
tags: ['toast', 'alert']
---

This component must be used with react-hot-toast and AlertProvider.

It displays a message, and optionally points earned by the user.

### Component usage
```js
import { Toast } from '@logora/debate.dialog.toast';
```

```js
<Toast 
    text={"Your text"}
    points={34}
    variant={"success"}
/>
```
