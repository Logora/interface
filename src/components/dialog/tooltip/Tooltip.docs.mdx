A component that renders a tooltip.
The text can be changed when the tooltip is clicked.

If no text is passed, then the component will only render the children.

### Component usage
```js
<Tooltip 
    text={"Information text"}
    onClickText={"Other information"}
    position={"top"}
>
    Hover and Click !
</Tooltip>
```
