A box component that displays the title, user progress, description and image of a badge.

### Component usage

```jsx
import { BadgeBox } from '@logora/debate.gamification.badge_box';
```

```jsx
<BadgeBox 
    badge={{badge: {name: "Influencer"} }}
    eloquenceTitle={"Influencer"}
/>
```
