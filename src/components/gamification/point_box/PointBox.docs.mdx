A box component with an icon, some text and time ago to display points.

### Component usage

```jsx
import { PointBox } from '@logora/debate.gamification.point_box';
```
```jsx
<PointBox 
    icon={svg icon}
    text={"Your text"}
    timeAgo={"12 days ago"}
/>
```
