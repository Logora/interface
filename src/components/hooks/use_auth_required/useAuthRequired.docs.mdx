This hook dispatch a custom event to signal authentication.

### Component usage
```js
import { useAuthRequired } from '@logora/debate.hooks.use_auth_required';
```
```js
const Component = () => {
	const requireAuth = useAuthRequired();

	return (
        <>
            {requireAuthentication({})}
        </>
    )
}
```
