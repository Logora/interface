IconProvider is a context component to store a library of icons loaded dynamically.

Pass a library name, and it will load it and store it in the context.

Library module can be for example : `@logora/debate.icons.regular_icons`.

It should be used with the [Icon component](https://bit.cloud/logora/debate/icons/icon).


### Component usage

```js
import { IconProvider, IconContext } from '@logora/debate.icons.icon_provider';
```

```js
const ComponentWithIcons = () => {
	return <div>Hello world !</div>
}

<IconProvider libraryName={"@logora/debate.icons.regular_icons"}>
    <ComponentWithIcons />
</IconProvider>
```
