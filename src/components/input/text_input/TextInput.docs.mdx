import { useState } from 'react';
import { TextInput } from './TextInput';

### Usages

TextInput component with error and success props.  
Supports all props from native html input.  
Size support with em, the container of the component can use `font-size` to change the size of the component.

TextInput example:

```js
import { TextInput } from '@logora/debate.input.text_input';
```

```js live
() => {
  const [value, setValue] = useState('');
  return (
    <TextInput error={value === 'error'} placeholder="placeholder" success={value === 'success'} onChange={(e) => setValue(e.target.value)} />
  );
};
```

TextInput with icon on the right side:

```js live
<TextInput
  placeholder="search..."
  iconRight={
    <img
      src="https://static.bit.dev/bit-icons/filter.svg"
      style={{ cursor: 'pointer' }}
      onClick={() => alert('on icon click')}
    />
  }
/>
```

TextInput with icon on the left side:

```js live
<TextInput
  placeholder="search..."
  iconLeft={
    <img
      src="https://static.bit.dev/bit-icons/filter.svg"
      style={{ cursor: 'pointer' }}
      onClick={() => alert('on icon click')}
    />
  }
/>
```
