import { Toggle } from './Toggle';

A switch toggle component with all the native options, based on [checkbox from base-ui](https://bit.cloud/logora/debate/input/checkbox/label) scope.  
The size is based on `em` so we can change the size by changing the font-size.  
Basic example:

```js
import { Toggle } from "@logora/debate.input.toggle"
```
```js live
<Toggle />
```

Different size example:

```js live
<Toggle style={{ fontSize: 32 }} />
```
