A toggle with two labels.

### Component usage
```js
import { TogglePosition } from '@logora/debate.input.toggle_position';
```
```js
    <TogglePosition 
        activeLabel={0}
        firstLabel={"Yes"}
        secondLabel={"No"}
        onChange={(label) => label}
    />
```
