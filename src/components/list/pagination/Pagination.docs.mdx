A component that renders a button used to handle pagination of lists. 

### Component usage

```js
    import { Pagination } from "@logora/debate.list.pagination";
```
```js
    <Pagination
        buttonText={"Next page"}
        currentPage={1}
        perPage={10}
        totalElements={20}
        onLoad={loadMoreElementsCallback}
        isLoading={isLoadingVar}
        hideLoader={false}
    />
```

#### Props
- `currentPage`: Number of the current page.
- `perPage`: Number of elements per page.
- `totalElements`: Number of elements in total.
- `buttonText`: Text displayed on the button.
- `onLoad`: Callback function triggered when clicking on the pagination button.
- `isLoading`: Boolean to control the loading state.
- `hideLoader`: If `true`, will not show the loader on load.
