import { NavbarButton } from './NavbarButton';

A button that opens the navigation modal (NavbarModal). It is intended for mobile usage or inside a drawer. When not rendered inside a drawer (`inDrawer = false`), it only appears on mobile screens (width <= 576px). Inside a drawer (`inDrawer = true`), visibility can be controlled with `showInDrawer`.

### Props
- inDrawer (boolean) — renders the button styled for a drawer context.
- showInDrawer (boolean) — if false while `inDrawer` is true, the button is hidden.

### Component usage

To import the component:

```js
import { NavbarButton } from '@logora/debate.navbar.navbar_button';
```

To use the NavbarButton component:

```jsx
<NavbarButton />
```

In drawer:
```jsx
<NavbarButton inDrawer />
```

Hidden in drawer:
```jsx
<NavbarButton inDrawer showInDrawer={false} />
```
