import { ReportBox } from './ReportBox';

The ReportBox component is used to display a single report, including its classification, status (accepted, pending, or rejected).


### Component usage

```js
import { ReportBox } from '@logora/debate.report.report_box';
```

```js
const report = {
  id: 1,
  classification: 'Spam',
  is_processed: true,
  reportable_type: 'Message',
  reportable: {
    id: 84,
    content: 'This is a reported message.',
    author: { full_name: 'John Doe' },
    created_at: '2025-05-06T15:43:17.545Z',
    status: 'pending',
  },
  author: {
    full_name: 'Jane Smith',
  },
};

<ReportBox report={report} />
```
