import { SectionBox } from './SectionBox';

A component that renders a box with a styled title and optional collapsible behavior. Content has been passed as a child.

### Component usage
```js
<SectionBox 
    title="Section Title"
    subTitle="Section sub title"
    isCollapsedByDefault={true}
>
    Content
</SectionBox>
```
