import { SectionBox } from './SectionBox';

A component that renders a box with a styled title. Content has been passed as a child.

### Component usage
```js
<SectionBox 
    titleText="Section Title"
>
    Content
</SectionBox>
```

### Properties
| **Name**      | **Type** | **Required** | **Default** | **Description** |
| ----------- | ----------- | :-----------: | ----------- | ----------- |
| <font color='#4646c6'>titleText</font> | <font color="#397300">string</font> | &cross; | - | <font color="grey">*text for section's title*</font> |
