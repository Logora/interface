description: A user content skeleton component.
---

A skeleton for user content loading. The height of the body can be controlled with the `numberLines` prop.

### Component usage
```js
import { UserContentSkeleton } from '@logora/debate.skeleton.user_content_skeleton';
```
```js
<UserContentSkeleton>
    <div>Loading</div>
</UserContentSkeleton>
```
