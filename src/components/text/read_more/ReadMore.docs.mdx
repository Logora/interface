---
description: A component to truncate text with a read more button.
---

A component that displays truncated text with the possibility to go to the full text version with a link or can be expanded to show the full text.


### Component usage

```js
import { ReadMore } from '@logora/debate.text.read_more';
```

```js
    <ReadMore
        content={"My super long content"}
        lineCount={4}
        to="/link/to/read/more"
        className="MyClassName"
        readMoreText="Read more"
        readLessText="Read less"
    />
```
