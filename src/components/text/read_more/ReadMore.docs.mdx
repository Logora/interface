import { ReadMore } from './ReadMore';

A component that displays text and can be expanded to show the full text. This component does not accept any children as props and is an orphan component.

It should be wrapped with react-intl's IntlProvider

### Component usage
```js
import { ReadMore } from '@logora/debate.text.read_more';
```

```js
<IntlProvider locale="en">
    <ReadMore
        content={"My super long content"}
        contentCharCount={250}
        readMoreUrl="/link/to/read/more"
        nextLine={false}
        lineCount={false}
        nextLineSpacing={0}
        className="MyClassName"
    />
</IntlProvider>
```

### Properties
| **Name**      | **Type** | **Required** | **Default** | **Description**
| ----------- | ----------- | ----------- | ----------- | ----------- |
| <font color='#4646c6'>contentCharCount</font> | <font color="#397300">number</font> | &cross; | 250 | <font color="grey">*maximum character count to be displayed*</font> |
| <font color='#4646c6'>readMoreUrl</font> | <font color="#397300">string</font> | &cross; | - | <font color="grey">*url to be used for the read more link. The read more button acts as a toggle for the remaining content*</font> |
| <font color='#4646c6'>nextLine</font> | <font color="#397300">boolean</font> | &cross; | false | <font color="grey">*whether the read more text should be displayed on the next line*</font> |
| <font color='#4646c6'>lineCount</font> | <font color="#397300">boolean</font> | &cross; | false | <font color="grey">*whether the content should be limited to a certain line count instead of character count*</font>
| <font color='#4646c6'>nextLineSpacing</font> | <font color="#397300">number</font> | &cross; | 0 | <font color="grey">*spacing between the collapsed text and the read more link*</font> |
| <font color='#4646c6'>className</font> | <font color="#397300">string</font> | &cross; | - | <font color="grey">*className to be applied to the read more button text*</font> |
| <font color='#4646c6'>rest</font> | <font color="#397300">Object</font> | &cross; | - | <font color="grey">*rest of props that should be passed to the read more link*</font> |
