import { Dropdown } from './Dropdown';

A component that renders a dropdown component which first child is the button triggering the display of the list (second child passed to component).
Dropdown is closed when user clicks outside it. A click on any child from the list also close the list. Props can be used for specific display.

### Component usage
```js
    <Dropdown dropdownListRight={true}>
        {/* First child: trigger button */}
        <div>Title</div> 
        {/* Second child: list */}
        <div>
            <p>Child 1</p>
            <p>Child 2</p>
        </div>
    </Dropdown>
```

| **Name**      | **Type** | **Required** | **Default** | **Description** |
