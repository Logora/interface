A component to enable scroll based on page hash.
Wrap your component with HashScroll and pass an element ID.
If hash matches the elementId passed in props, this component will automatically trigger a scroll to the element.

It must be inside a router from react-router.

### Component usage

```js
    import { HashScroll } from '@logora/debate.tools.hash_scroll';
```

```js
    <BrowserRouter>
        <HashScroll 
            elementId={"myId"}
            onScroll={() => console.log("Starting scroll !")}
        >
            <div id="myId">Scroll to element</div>
        </HashScroll>
    </BrowserRouter>
```
