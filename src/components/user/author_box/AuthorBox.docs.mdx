import { AuthorBox } from './AuthorBox';

The AuthorBox component is used to display a user's avatar image, name link, points and occupation in a box.
It uses [ConfigProvider](https://bit.cloud/logora/debate/data/config_provider) for routing.
The avatar image is rendered using the [Avatar](https://bit.cloud/logora/debate/user/avatar) component.

It should be wrapped with react-intl's IntlProvider, ConfigProvider and be inside a Router.

### Component usage

```js
import { AuthorBox } from '@logora/debate.user.author_box';
```

```
import { AuthorBox } from './AuthorBox';

const author = {
  hash_id: 'john-doe',
  full_name: 'John Doe',
  image_url: 'https://example.com/avatar.jpg',
  points: 53
};

<AuthorBox 
  hashId={author.hash_id}
  fullName={author.full_name}
  avatarUrl={author.image_url}
  points={author.points}
/>
```
