A modal component that allows users to select his first name, last name and language.

It must be inside [IconProvider](https://bit.cloud/logora/debate/icons/icon_provider).

### Component usage

```jsx
import OnboardingModal from '@logora/debate.user.onboarding_modal';

<ModalProvider>
    <IntlProvider locale="en">
        <AuthProvider>
            <OnboardingModal />
        </AuthProvider>
    </IntlProvider>
</ModalProvider>
