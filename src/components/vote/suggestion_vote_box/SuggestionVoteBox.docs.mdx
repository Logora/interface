A button allowing users to upvote or downvote on items.

Button can be disabled if needed.

### Component usage

```js
import { SuggestionVoteBox } from '@logora/debate.vote.suggestion_vote_box';
```

```js
<SuggestionVoteBox
    voteableType={"Message"}
    voteableId={45}
    totalUpvote={10}
    totalDownvote={0}
    disabled={false}
/>
```
